A simple, open source Riemann client for C#.

If an event is one-time, use SendEvent to push events to Riemann.

To send a value every n seconds, use the Tick method. riemann.cs will call you back every n seconds to get a new value, then send all of the events together. 


var client = new Riemann.Client("localhost", 5555);
var count = 0L;
client.Tick(10, "myservice.count", () => {
   var now = Interlocked.Exchange(ref count, 0L);
   return new TickEvent(now > 10 ? "warning" : "ok", "", now);
});
client.SendEvent("myservice.loaded", "ok", "Service loaded", 0.0);
